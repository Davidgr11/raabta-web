# Studially Website ✏️📱

## Información 📚
> Este proyecto consiste en la creación del website oficial de Studially mediante la implementación de conocimientos en **HTML y CSS**.

### Vista Previa 💡
Deploy en Netlify con dominio personalizado: [AQUÍ](https://studially.com/)

### Aspectos Técnicos 🛠️
* Mobile First
* BEM Architecture
* Responsive Design
* SASS, CSS preprocessor
* Animate CSS Library

![responsively](build/img/responsively.png)


### Local Installation 💻
> [!IMPORTANT]
> To install & run locally, you must take the next steps.
1. 
``` 
git clone "https://github.com/Davidgr11/studially-web.git"
npm init
```
2. Open with live server "index.html"
